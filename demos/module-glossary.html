Yunkoo Atlas · 模块词典

沟通时请优先用「建议统一名」。本页按产品真实结构拆解，不改正式代码。 点卡片看详情；筛选/搜索可快速对齐「我们说的是哪一块」。

1. 壳层

侧栏 + 主框 + 顶栏，所有页面共用。

2. 一级模块

今日记录 / 交易日志 / 案例记录 / 仪表盘。

3. 快捷 + 工作台

快捷入口、列表/看板/表格、筛选、快捷视图、详情。

4. 设置与浮层

策略/标签/数据等配置；命令面板、新建弹层、灯箱。

命名对齐提示： 「模拟回测」是侧栏/导航统一名；数据字段仍是 tradeKind: paper。 「交易日志」是一级模块名；进行中/星标/错过是快捷视图入口，不是新的一级模块。 「案例记录」≠ 旧「判例」；笔记本是案例模块内的子能力（设计已确认，实现进度以代码为准）。 改某模块时，请直接说统一名，例如：「改交易详情右侧属性栏」，而不是「改那个侧边栏」。
