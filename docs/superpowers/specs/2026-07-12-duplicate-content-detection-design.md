# 明显重复检测（正文 / 截图）

日期：2026-07-12

## 目标

拦住 Notion / CSV 反复导入造成的「同一条内容再进库」，并提供日常扫描清理。  
**不**把同日同品种同方向的多笔正常交易当成重复。

## 判定

- 正文指纹：去掉 HTML / 图片占位后的纯文本，长度 ≥ 24 且哈希相同 → 正文重复
- 截图指纹：图片字节 SHA-256；集合相同且 ≥ 2 张 → 截图重复
- 一张图 + 较短正文（≥ 8）同时相同 → 正文与截图重复
- 空内容、过短正文、单张无正文 → 不判重复

## 能力

1. Notion / CSV 导入预览：标出疑似重复，默认跳过，可「仍导入」
2. 设置 → 数据与导入 → 扫描库内重复 → 保留较新一条，其余进回收站
