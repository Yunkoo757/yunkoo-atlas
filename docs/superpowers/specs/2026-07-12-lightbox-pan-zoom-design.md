# 灯箱滚轮缩放 + 拖动画布

日期：2026-07-12  
状态：已确认（基于 HTML demo 选项）

## 目标

案例/交易正文双击图片打开灯箱后，支持：

1. **滚轮缩放**，锚点为**光标位置**
2. **左键拖动**平移画布（**100% 亦可拖**）
3. **点击遮罩空白始终关闭**（与是否放大无关）

## 非目标

- 不引入第三方 viewer 库
- 不做双指捏合专项优化（触控板滚轮缩放已覆盖常见场景）
- 本次不发布新版本

## 行为细则

| 操作 | 行为 |
|---|---|
| 滚轮 | 以光标为锚点缩放，范围约 25%–800% |
| 左键拖动 | 任意缩放级别均可平移 |
| 点击遮罩空白 | 始终关闭（拖拽后松手不视为点击） |
| 点击图片区域 | 不关闭（便于开始拖拽） |
| 双击画布 / 重置 | 回到 100% 并居中 |
| 切图 ← → | 重置变换 |
| Esc / 关闭按钮 | 关闭灯箱 |

## 实现落点

- `src/lib/lightboxView.ts` — 纯函数变换
- `src/components/ImageLightbox.tsx` + `.css` — UI 与指针事件
- 保留既有快捷键 scope `lightbox`
